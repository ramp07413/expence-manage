# **App Name**: Pastel Ledger

## Core Features:

- Expense Tracking: Allows users to log and categorize their expenses.
- Budget Management: Enables users to set and manage budgets across different categories.
- Report Generation: Generates visual reports of spending habits.
- Smart Category Suggestions: Uses AI to suggest appropriate expense categories.

## Style Guidelines:

- Primary color: Pale blue (#B0E2FF) for a calm, trustworthy feel.
- Background color: Very light blue (#F0F8FF) for a clean, minimal aesthetic.
- Accent color: Soft lavender (#E6E6FA) for highlights and interactive elements.
- Body and headline font: 'Inter', a sans-serif, for a modern, objective feel.
- Simple, outlined icons to represent expense categories.
- Clean, responsive grid layout with a focus on whitespace.
- Subtle transitions and animations for a smooth user experience.
